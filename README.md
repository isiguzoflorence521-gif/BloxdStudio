# 🎨 BloxdStudio

**BloxdStudio** is a web-based, unofficial development tool designed for [Bloxd.io](https://bloxd.io) creators. It provides a familiar, studio-like environment to build maps, edit 3D models, and manage game assets.

---

## 🚀 Features

- 🧱 **Map Builder**: Create 3D structures and edit terrain with ease.
- 🎨 **3D Model Editor**: Import, preview, and re-texture 3D objects.
- 📁 **Schematic Support**: Save and load maps using the `.bloxdschem` format.
- 🌐 **Web-Based**: No installation required—runs directly in your browser.

## 🛠️ Getting Started

### Prerequisites
You only need a modern web browser (Chrome, Firefox, or Edge recommended).

### Usage
1. Visit the [Live Demo](https://isiguzoflorence521-gif.github.io/BloxdStudio/
).
2. Use the **Build** tab to start placing blocks.
3. Export your work via the **File** menu to save your `.bloxdschem` file.

## ⚙️ Development

This project is built using:
- **HTML5/CSS3** for the user interface.
- **JavaScript** for core logic and 3D rendering.
- **Three.js** (planned) for high-performance 3D visualization.
